Catalogue des options de compilation
Les options de compilation occupent 5 lignes au début du programme .ino
Fichier: OptionsCompilation.txt

Section 1: Compilation par Arduino IDE  (ne sert plus)

====Pour ESP32 MH+ET partition 1Mb prog 1Mb OTA et 2 Mb FileSystem
// python-build-start
// /opt/arduino-1.8.13/arduino
// esp32:esp32:mhetesp32minikit:FlashFreq=80,PartitionScheme=default --preserve-temp-files
// /dev/ttyUSB0
// python-build-end
 
====Pour ESP32 MH+ET partition 1.9Mb prog 1.9Mb OTA et 1.9 Kb FileSystem (Appli BLE)
// python-build-start
// /opt/arduino-1.8.13/arduino
// esp32:esp32:mhetesp32minikit:FlashFreq=80,PartitionScheme=min_spiffs,UploadSpeed=921600,DebugLevel=none
// /dev/ttyUSB0
// python-build-end

====Pour ESP8266 D1 Mini partition 1Mb prog 1Mb OTA et 2 Mb FileSystem

// python-build-end
